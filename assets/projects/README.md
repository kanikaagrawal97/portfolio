# Project Assets

This directory is where you should place your project screenshots or logos.

## How to add images:

1.  Save your image in this folder: `public/assets/projects/`
2.  Name the file exactly as it appears in your `src/constants/constants.ts` file under the `thumbnail` property.

## Example:

If you have a project in `constants.ts` like this:

```typescript
{
  title: "Git Pull Watched",
  thumbnail: "git-pull-watched.png"
}
```

You should save your image as `public/assets/projects/git-pull-watched.png`.

## Supported Formats:

- PNG (.png)
- JPEG (.jpg, .jpeg)
- SVG (.svg)
- WebP (.webp)
